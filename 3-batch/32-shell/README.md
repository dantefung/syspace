# 标准shell脚本包含哪些元素

```
bash demo2.sh
```
该执行方式是可以没有执行权限也能执行.
会产生新的子进程。
```
./demo2.sh
```
需要执行权限。
会产生子进程。
```
chmod u+x demo2.sh
```
赋予执行权限

```
source ./demo2.sh
```
这个会对当前的执行环境产生影响。
```
.demo2.sh
```

# 管道与管道符
- 管道和信号一样，也是进程通信的方式之一。
- 匿名管道（管道符）是shell编程经常用到的通信工具。
- 管道符是“|”,将前一个命令的执行结果传递给后面的命令。

		e.g.  cat | ps -f
