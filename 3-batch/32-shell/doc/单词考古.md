---

## 🌀 Profile 词义演化图

```
词源 (英文单词)
│
├── 原始含义
│     ├── 侧面像 / 轮廓 (outline)
│     ├── 简介 / 描述 (summary)
│     └── 画像 (portrait)
│
└── 软件工程中的衍生含义
      │
      ├── 配置环境 Profile
      │     - Spring Profiles (dev / test / prod)
      │     - 定义运行环境的“轮廓”
      │
      ├── 性能分析 Profile
      │     - Profiler 工具生成的报告
      │     - 程序运行时的“性能画像”
      │
      ├── 建模扩展 Profile
      │     - UML Profile (SysML, MARTE)
      │     - 对标准 UML 的“定制化扩展”
      │
      └── 用户画像 Profile
            - 用户属性与行为的抽象
            - 用户的“数字化侧面像”
```

---

### 📌 关键点

* **核心隐喻**：从“轮廓/画像” → 扩展为“某个对象在特定维度下的抽象”
* **跨领域应用**：工程配置、运行性能、模型扩展、用户数据，都可以用 profile 来描述
* **术语化**：在软件工程里说 “profile” 时，不需要翻译回字典义，而是直接理解为“某个维度下的画像/轮廓”
